# Gerenciador de Finanças

Este é um site para gerenciamento de finanças pessoais, desenvolvido usando HTML, CSS, JS e PHP.

# Funcionalidades
O site oferece as seguintes funcionalidades:

* Cadastro de despesas e receitas, com categorias e datas.
* Listagem de despesas e receitas cadastradas, com filtros por categoria e data.
* Edição e exclusão de despesas e receitas cadastradas.
* Dashboard com gráficos e estatísticas de finanças.
